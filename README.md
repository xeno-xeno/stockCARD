# Stock Ticker Quiz 🎯

미국 주식의 티커(Ticker Symbol)를 보고 올바른 회사명을 맞추는 퀴즈 게임입니다.

## 🎮 Live Demo

GitHub Pages에서 바로 플레이하세요: [Stock Ticker Quiz](https://xeno-xeno.github.io/stockCARD/)


## 🎯 게임 방법

1. 화면에 표시되는 주식 티커(예: AAPL)를 확인
2. 3개의 회사명 중 올바른 답을 선택
3. 연속으로 맞춰서 최고 스트릭 달성
4. 팀원들과 점수 경쟁!

## 🏗️ 기술 스택

- **React 18** - UI 프레임워크
- **Vite** - 빌드 도구 및 개발 서버
- **Tailwind CSS** - 스타일링
- **Lucide React** - 아이콘
- **GitHub Pages** - 호스팅
